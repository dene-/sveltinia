# Installation

::: code-group

```sh [Yarn]
yarn add sveltinia
```

```sh [npm]
npm install sveltinia
```

```sh [pnpm]
pnpm add sveltinia
```

```sh [Bun]
bun add sveltinia
```

:::

Choose whichever package manager your project already uses. Sveltinia supports Svelte 4 and Svelte 5.
